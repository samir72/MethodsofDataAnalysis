Assignment # 8 : Time Series Analysis.¶

Data Science 350¶

In this assignment I am performing time series analysis on both Milk and Ice cream production data provided in CADairyProduction.csv. My goal here is to validate if these time series are stationary, whether they have significant seasonal impact, decompose the time series and find the best ARIMA fit on the residual and then finally forecast the production numbers on both Milk and Ice cream for year 2014.

  • Following libraries have been used for this assignment.
    • stats
    • forecast
  • At the start we load the data file, a cursory review tells you that it would be best analysed by time series objects.
  • We now use the ts function to initially create six(Transformed and untransformed Milk,Ice Cream & Cottage Cheese data) time series objects to plot this data.
    • We immediately get to see the following in the plot.
      • Milk Data
        • A clear upward trend with seasonal aspect, a slight dip in 2010 as compared to the previous years.
      • Ice Cream Data
        • Significant seasonal aspect.
      • Cottage cheese.
        • A clear downward trend which at the outset also looks like a 'Random Walk'.
  • From this point forward, I will now be only looking at Milk and Ice Cream data. Let us now check the distribution and QQ plot of both Milk and Ice cream data.
    • Milk Data
      • Both transformed and untransformed data looks normal with long tails.
    • Ice Cream Data
      • Both transformed and untransformed data looks normal with long tails.
  • Check ACF and PACF for both Milk and Ice Cream time series object.
    • Milk : This time series is clearly NOT stationary *with decent trend and seasonality*.
      • For both transformed and untransformed data ACF is decaying slowly and PACF is showing values outside the confidence intervals for atleast one lag, this would point to a trend creating dependancy between consecutive values.
    • Ice Cream : This time series seems to be STATIONARY as its decaying over time, i.e. it seems to be stable over the time period and has *strong seasonality*.
      • For both transformed and untransformed data once see clear oscillations in ACF at regular intervals. In this case between 0 and 1(One year) you can see 12 spikes.
  • I am using multiplicative model in decomposition of milk data as it has both trend and seasonality and additive model in decomposition of 'Ice Cream' data.
    • ACF plot of Milk data clearly shows a sudden drop when compared to a long drawn out delay on original data. This would mean that trend data has been removed.
    • ACF plot of Ice Cream data shows a value of 1 at lag 0 and then flattens out, giving us a clear indication that seasonlity has now been removed from this data.
  • Residual obtained from this decomposition is then used to obtain the optimal ARIMA model using arima function and then further validated with auto.arima function from forecast package.
    • Given below are findings of Milk and Ice Cream data.
      • Milk : ARMA(p,q) = 0,1 ; ARIMA(p,q,d) = 0,1,1, here "d" of 1 further confirms the non-stationary series as a difference of a 1 was required to come up with a optimal season plan.
      • Ice Cream : ARMA(p,q) = 3,1 ; ARIMA(p,q,d) = 3,0,1, here "d" of 0 further confirms the stationary series as no difference was required to make the data stationary.

#check the distribution
dist.ts = function(df, col = 'residual', bins = 40){
  par(mfrow = c(1,2))
  temp = as.vector(df)
  breaks = seq(min(temp), max(temp), length.out = (bins + 1))
  hist(temp, breaks = breaks, main = paste('Distribution of ', col), xlab = col)
  qqnorm(temp, main = paste('Normal Q-Q plot of ', col))
  par(mfrow = c(1,1))
}

#plot the autocorrelation function (acf) and partial autocorrelation function (pacf) of the white noise series
plot.acf <- function(df, col = 'remainder', is.df =TRUE, decomp = TRUE){
  if(is.df) temp <- df[, col]
  else temp <- df
  par(mfrow = c(2,1))
  if(decomp)
  {
    acf(temp, main = paste('Post decomposition ACF of', col))
    pacf(temp, main = paste('Post decomposition PACF of', col))
  }
  else
  { 
    acf(temp, main = paste('Pre decomposition ACF of', col))
    pacf(temp, main = paste('Pre decomposition PACF of', col))
  } 
  par(mfrow = c(1,1))
}

## Decomposition of the time series into components
ts.decomp <- function(df, col = 'elec.ts', span = 0.5, Mult = TRUE, is.df = TRUE){
  #if(Mult) temp = log(df[, col])  else temp = ts(df[, col]
  if(is.df) temp = log(df[, col])  
  else temp = df
  spans = span * length(temp)  
  fit <- stl(temp, s.window = "periodic", t.window = spans)
  plot(fit, main = paste('Decompositon of',col,'with loess span = ', as.character(span)))
  fit$time.series
}

#Apply ARIMA Model
## Function for ARIMA model estimation
ts.model = function(ts, col = 'remainder', order = c(0,0,1)){
  mod = arima(ts, order = order, include.mean = FALSE)
  print(mod)
  mod
}
